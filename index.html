A simple cross-platform file transfer client with support for multiple protocols and direct transfer between remote hosts.

Features highlight:

  • Easy to use, slim interface
  • Support for multiple protocols: FTP, SFTP, Windows Share (Samba), ...
  • Cross platform (works on Windows, Mac OS X and Linux)
  • Native look'n'feel on all platform
